GET /units/:id
Shows a specific unit for an army

Examples

// Shows a specific unit for an army
GET /armies/1139/units/34266
200
{
  "id": 34266,
  "type": "PikemanUnit",
  "strength": 5
}
>>>

POST /:id/train
Trains a Unit

Examples

// Does not find the unit to train
POST /armies/1143/units/34416/train
404
// Does not train the unit (not enough coins)
POST /armies/1146/units/34476/train
422
{
  "coins": [
    "must be greater than or equal to 0"
  ]
}
// Trains a Pikeman
POST /armies/1141/units/34326/train
200
{
  "id": 34326,
  "type": "PikemanUnit",
  "strength": 8
}
>>>

POST /:id/transform
Transforms a Unit

Examples

// Does not find the unit to transform
POST /armies/1149/units/34597/transform
404
// Does not transform the unit (not enough coins)
POST /armies/1152/units/34657/transform
422
{
  "coins": [
    "must be greater than or equal to 0"
  ]
}
// Does not transform the knight
POST /armies/1154/units/34746/transform
422
{}
// Transforms a pikeman
POST /armies/1147/units/34506/transform
200
{
  "id": 34506,
  "type": "PikemanUnit",
  "strength": 5
}
