GET /units/:id
Shows a specific unit for an army

Examples

// Shows a specific unit for an army
GET /armies/1166/units/35079
200
{
  "id": 35079,
  "type": "PikemanUnit",
  "strength": 5
}
>>>

POST /:id/train
Trains a Unit

Examples

// Does not find the unit to train
POST /armies/1170/units/35229/train
404
// Does not train the unit (not enough coins)
POST /armies/1173/units/35289/train
422
{
  "coins": [
    "must be greater than or equal to 0"
  ]
}
// Trains a Pikeman
POST /armies/1168/units/35139/train
200
{
  "id": 35139,
  "type": "PikemanUnit",
  "strength": 8
}
>>>

POST /:id/transform
Transforms a Unit

Examples

// Does not find the unit to transform
POST /armies/1176/units/35410/transform
404
// Does not transform the unit (not enough coins)
POST /armies/1179/units/35470/transform
422
{
  "coins": [
    "must be greater than or equal to 0"
  ]
}
// Does not transform the knight
POST /armies/1181/units/35558/transform
422
{
  "type": [
    "Invalid transformation"
  ]
}
// Transforms a pikeman
POST /armies/1174/units/35319/transform
200
{
  "id": 1174,
  "type": "EnglishArmy",
  "coins": 1970
}
