>>>

POST /armies
Creates an army

Examples

// Creates an army
POST /armies
{
  "army": {
    "type": "EnglishArmy"
  }
}
201
{
  "id": 1163,
  "type": "EnglishArmy",
  "coins": 2000
}

Params

Param name Description
army
required

Validations:

  • Must be a Hash
army[type]
required

Validations:

  • Must be one of: EnglishArmy, ChineseArmy, ByzantineArmy.
