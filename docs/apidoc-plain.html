Armies

Armies / index

GET /armies
Lists all the armies

Examples

// Lists the armies
GET /armies
200

    "id": 1130,

Armies / show

GET /armies/:id
Shows a specific army

Examples

// Shows a specific army
GET /armies/1132
200
{
  "id": 1132,
  "type": "EnglishArmy",
  "coins": 2000
}

Armies / create

POST /armies
Creates an army

Examples

// Creates an army
POST /armies
{
  "army": {
    "type": "EnglishArmy"
  }
}
201
{
  "id": 1136,
  "type": "EnglishArmy",
  "coins": 2000
}

Params

  • army : required [

    Must be a Hash

    ]
    • type : required [

      Must be one of: EnglishArmy, ChineseArmy, ByzantineArmy.

      ]

Units / train

POST /:id/train
Trains a Unit

Examples

// Does not find the unit to train
POST /armies/1143/units/34416/train
404
// Does not train the unit (not enough coins)
POST /armies/1146/units/34476/train
422
{
  "coins": [
    "must be greater than or equal to 0"
  ]
}
// Trains a Pikeman
POST /armies/1141/units/34326/train
200
{
  "id": 34326,
  "type": "PikemanUnit",
  "strength": 8
}
