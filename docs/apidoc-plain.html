Armies

Armies / index

GET /armies
Lists all the armies

Examples

// Lists the armies
GET /armies

    "id": 1157,

Armies / show

GET /armies/:id
Shows a specific army

Examples

// Shows a specific army
GET /armies/1159
200
{
  "id": 1159,
  "type": "EnglishArmy",
  "coins": 2000
}

Armies / create

POST /armies
Creates an army

Examples

// Creates an army
POST /armies
{
  "army": {
    "type": "EnglishArmy"
  }
}
201
{
  "id": 1163,
  "type": "EnglishArmy",
  "coins": 2000
}

Params

  • army : required [

    Must be a Hash

    ]
    • type : required [

      Must be one of: EnglishArmy, ChineseArmy, ByzantineArmy.

      ]

Units / train

POST /:id/train
Trains a Unit

Examples

// Does not find the unit to train
POST /armies/1170/units/35229/train
404
// Does not train the unit (not enough coins)
POST /armies/1173/units/35289/train
422
{
  "coins": [
    "must be greater than or equal to 0"
  ]
}
// Trains a Pikeman
POST /armies/1168/units/35139/train
200
{
  "id": 35139,
  "type": "PikemanUnit",
  "strength": 8
}
